# award-api

API for awards APP

# Command

- start : `npm run start`
- start in dev : `npm dun dev`
- migrate db : `npm run migrate`
- seed db : `npm run seed

# How To Run

1. copy .env.example to .env and configure it
2. run migrate
3. run seed
4. start app

# Stack

- Node
- Express
- Mysql / MariaDB
